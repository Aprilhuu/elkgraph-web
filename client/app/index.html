ELK Graph Viewers

A selection of interactive editors and browsers to try automatic layout with.

Automatic layout algorithms compute layouts for diagrams. Never tried them? Well, this is your chance. Linked below you will find a number of editors or browsers driven by automatic layout. Use the editors to specify your own diagrams to be laid out, or use the browser to look at what automatic layout does to a number of test diagrams we provide.

Everything you see here is based on the following technologies:

  • sprotty
    A next-gen web-based graphics framework primarily developed by TypeFox.
  • Eclipse Layout Kernel (ELK)
    Java-based implementations of automatic layout algorithms.
  • Monaco Editor
    The code editor that powers Visual Studio Code.
  • LSP4J
    A Java binding for the Language Server Protocol.
  • Xtext
    A framework for development of programming languages and domain-specific languages.
